## HTML code blocks

> HTML code block will be loaded as a Vue component by default.

```html
<p class="html-code-block">This is a code block of HTML.</p>

<style>
.html-code-block {
  border-radius: .25rem;
  border: 1px solid #eee;
  margin-bottom: 1rem;
  padding: .5rem 1rem;
}
</style>
```
